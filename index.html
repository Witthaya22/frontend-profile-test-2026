ผมเป็นนักศึกษาปีที่ 1 ผมเป็นนักศึกษาเทคโนโลยีสารสนเทศเพื่อการจัดการอุตสาหกรรม

ประวัติของฉัน:

  • เกิดวันที่ 22 กันยายน 2547
  • จบการศึกษาประกาศนียบัตรวิชาชีพชั้นสูงจากวิทยาลัยเทคนิคชัยภูมิ
  • ปัจจุบันกำลังศึกษาอยู่ที่มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าเจ้าพระนครเหนือ

เป้าหมาย:

ผมต้องการพัฒนาทักษะและเป็นนักพัฒนาเว็บไซต์ที่มีความเชี่ยวชาญเพื่อสร้างสรรค์งานที่มีคุณค่า

ผลงานของฉัน

ระบบจัดการข้อมูลสำหรับการแข่งขันสิ่งประดิษฐ์

รับหน้าที่ออกแบบและพัฒนาเว็บไซต์สำหรับแสดงผลและจัดการข้อมูลที่ส่งมาจาก ระบบ IOT ตรวจเช็คการใช้งานไฟฟ้า ของโครงงาน และการสื่อสารข้อมูลแบบเรียลไทม์ผ่านหน้าเว็บ

เทคโนโลยีที่ใช้: Nuxt.js, Express.js, REST API, MySQL

ดูผลงานบน GitHub

ระบบจัดการกิจกรรมวิทยาลัยเทคนิคชัยภูมิ โปรเจคต์จบ

รับหน้าที่ออกแบบและพัฒนาเว็บไซต์สำหรับจัดการกิจกรรมต่างๆ ภายในวิทยาลัยเทคนิคชัยภูมิ เช่น การลงทะเบียนกิจกรรม การตรวจสอบสถานะการเข้าร่วมกิจกรรม และการแสดงผลข้อมูลกิจกรรมต่างๆ

เทคโนโลยีที่ใช้: Nuxt.js, Express.js, REST API, PostgreSQL
